# Axis-App

Axis is a life management & companion app that makes it easy to control everything in the daily life, from simple Todo list, to finance and health monitors, all in one place.


## Features:
- 📃 Todo List
- ⚖ Weight Monitor
  - 📊 Weekly Weight Chart
- 🎨 Custom Themes


## Upcoming Features:
- 📅 Day Planner
- 🧮 Financial Monitor
- 🔥 Calories Monitor
- ⚖ Weight Monitor
  - 📊 Monthly Weight Chart
- 💧 Period Tracker
